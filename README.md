# OpenUnum

OpenUnum is a lightweight, Ubuntu-first personal agent framework focused on:
- strong tool calling
- multi-provider model routing
- memory
- browser automation via remote debugging (CDP)
- WhatsApp + Telegram channels
- clean web + mobile web UI
